Property Detail
aliasIdproperty
public var aliasId:String = __jsonclass__

The property to use to identify the remote class identifier. By default, this value is "__jsonclass__", which is consistent with json-rpc, but it can be re-set here to anything.

decodeErrorHandleModeproperty 
public var decodeErrorHandleMode:String = DECODE_ERROR_TRACE

How should Argonaut behave when it runs into errors while decoding? By default, we simply catch and trace the errors (equivalent to AMF). Other options: ArgonautConstants.DECODE_ERROR_ERROR - Throw errors ArgonautConstants.DECODE_ERROR_IGNORE - Do nothing at all, not even trace

nativeEncodeModeproperty 
public var nativeEncodeMode:Boolean = false

If set to true, encoding uses AS-native JSON.stringify() Native mode is internally faster, but won't respect [DontSerialize] tags and won't automatically add an alias tag.

tagClassesWhenEncodingproperty 
public var tagClassesWhenEncoding:Boolean = true

By default, all complex objects get tagged with the aliasId during serialization. Set this to false to suppress tagging.
