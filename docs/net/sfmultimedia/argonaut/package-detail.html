Classes
 ClassDescription
 Argonaut Argonaut provides AMF-style class serialization/deserialization between AS3 and JSON objects.
 ArgonautConfig These values can be overridden to change Argonaut behavior
 ArgonautConstants Constants used internally by Argonaut. The public DECODE_ERROR constants may be set on a configuration in order to change error reporting behavior.
 ArgonautErrorEvent Error event thrown by Argonaut classes.
 ClassRegister Responsible for registering classes and mapping their non-static properties. Classes may be mapped because they are being shared across the application boundary, or simply in the process of serialization. Note that some data types get "normalized" to ensure clean reflection to JSON data types.
 ErrorHandler Refer all errors to this class.
 JSONDecoder Responsible for converting raw JSON into Classes
 JSONEncoder Takes any AS instance and convert its public, non-static properties to JSON. Essentially, this is just a souped up version of JSON.stringify(), adding two features: Classes may specify a [DontSerialize] metatag so that you can suppress properties you don't wish to serialize.
 PropertyTypeMapping Mapping record of each of a class's properties
